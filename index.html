What Can It Do?

This bot helps you keep your Discord server clean and fun with powerful moderation tools and entertaining commands.

Moderation Commands

  • !ban [user] - Bans a user
  • !kick [user] - Kicks a user
  • !mute [user] - Mutes a user
  • !warn [user] - Issues a warning
